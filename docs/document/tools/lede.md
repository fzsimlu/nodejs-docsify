## Lede中的常用软件


### 网络共享-samba

- 登陆路由器-网络存储-网络共享-编辑模板

#注释点下面代码   
#invalid users = root

- 设置用户/密码

```bash
$ smbpasswd -a root
# 输入 123456
```

- 共享目录

| 名称 | 目录 |  允许用户  |只读  |可预览  |允许匿名用户|创建权限掩码|目录权限掩码|
|-------| -------| -------| -------| -------| -------| -------| -------|
| gd      | /mnt/gd   |   root     ||√||0775|0775|

- 重启路由器完成

- windows 检测是否开启smb协议
- windows 打开网络，找到共享目录输入上面设置的账号密码
